# Example Results

This folder contains results (fiber dynamics and turbulence statistics) generated using the code from a representative simulation in a cubic unit cell ($H = 40$ and $N = 128$) with Taylor-scale Reynolds number $\mathrm{Re} = 55$ and $k_{max} \eta = 3.5$, where $k_{max} = \frac{2\pi}{H} \frac{\sqrt{2}N}{3}$ is the highest resolved wavenumber. The turbulent flow is allowed to develop first and a statistical stationarity is reached. Next, 8 fibers with lengths $L$ such that $L/\eta = 11$ are introduced into the box. The time instant at which fibers are introduced is denoted by a red vertical line in the plots for turbulent flow statistics.
